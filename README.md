# DST Framework

The DST Framework is designed to provide insights into dynamic systems through detection mechanisms tailored for specific regimes and model behavior. This framework includes key components for understanding and analysis:

## Model
The framework allows for various model configurations that enable effective detection and analysis of dynamic systems, focusing on key performance indicators and characteristics unique to each use case.

## Regimes
The DST Framework operates under specific regimes that dictate its behavior. Each regime is configured to respond to different system dynamics, ensuring accurate detection and response to changes in the environment.

## What It Detects
The framework is capable of detecting:
- Anomalies in system behavior
- Changes in performance indicators
- Deviations from expected trajectories

## Installation
To install the DST Framework, follow the steps below:
1. Clone the repository:
   ```bash
   git clone https://github.com/idanreph/dst-framework.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dst-framework
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
After installation, you can start using the DST Framework by importing it into your project and initiating detection processes as per the documentation provided in the repository.
